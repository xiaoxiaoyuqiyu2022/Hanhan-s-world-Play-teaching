| 指令              | 说明                |
| --------------- | ----------------- |
| /setwarp \[地标名] | 创建名为\[地标名]的地标     |
| /delwarp \[地标名] | 删除名为\[地标名]的地标     |
| /warp \[地标名]    | 将自己传送到名为\[地标名]的地标 |
| /warps          | 打开地标列表GUI         |
